一、 测试用例设计规则
1. 命名规范
1.1.项目名称
首字母大写+大写式驼峰，
ProjectName

1.2.模块名和包名
全部小写+下划线驼峰
module_name  package_name

1.3.类名称，异常
首字母大写+大写式驼峰，
class ClassName:   ，ExceptionName

1.4.全局变量、常量
全部使用大写字母+下划线驼峰
GLOBAL_VAR_NAME，CONSTANT_NAME

1.5.方法名，函数名，其余变量，参数，实例
全部小写+下划线驼峰
method_name，function_name，instance_var_name, function_parameter_name, local_var_name

1.6.处理计数器外，不使用单字母命名
